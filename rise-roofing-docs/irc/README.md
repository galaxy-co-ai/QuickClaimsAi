# International Residential Code (IRC) References

Code documentation used for building code-based arguments.

## Folder Structure

```
irc/
├── 2018/     # IRC 2018 edition
├── 2021/     # IRC 2021 edition (most common)
└── 2024/     # IRC 2024 edition
```

## What Goes Here

- IRC chapter excerpts (especially Chapter 9: Roof Assemblies)
- State-specific amendments
- Code commentary documents
- Reference tables (R905, R906, etc.)

## Priority Sections

### Chapter 9 - Roof Assemblies (Most Important)
- R905.1 - Roof covering materials
- R905.2 - Asphalt shingles
- R905.7 - Wood shingles/shakes
- R905.8 - Metal roof shingles

### Chapter 3 - Building Planning
- R301.2 - Climatic and geographic design criteria
- R309 - Ice barrier requirements

## File Naming

```
[section-number]-[section-name].pdf
```

**Examples:**
- `R905-roof-coverings.pdf`
- `R301-climatic-design-criteria.pdf`
- `chapter-09-complete.pdf`

## How AI Uses These

1. Auto-lookup code requirements for argumentation
2. Region-specific code requirement identification
3. Building code-based supplement justification
4. Counter-argument support when carrier disputes

## Notes

- Put files in the correct version folder (2018, 2021, or 2024)
- Most states currently use 2021 IRC
- Check `/regions/` for state-specific IRC adoption
