# Past Scopes

Historical supplement scopes used for AI pattern learning and training.

## What Goes Here

- Completed supplement documents (approved or denied)
- Full scope PDFs from insurance carriers
- Before/after comparison documents
- Successful argumentation examples

## File Naming

```
YYYY-MM_[homeowner-last]_[carrier]_[outcome].pdf
```

**Examples:**
- `2025-01_johnson_state-farm_approved.pdf`
- `2024-11_martinez_allstate_partial.pdf`
- `2024-08_williams_travelers_denied.pdf`

## Outcomes
- `approved` - Full supplement approved
- `partial` - Partial approval
- `denied` - Denied (useful for learning what doesn't work)

## Privacy Requirements

**BEFORE UPLOADING:**
1. Redact full homeowner name (last name initial OK)
2. Redact full address (city/state OK)
3. Redact claim number
4. Redact policy number
