# Standard Roof Supplement Checklist

> Use this checklist when reviewing insurance scopes to identify missing items.

---

## ðŸ”´ REQUIRED ITEMS (Check Every Scope)

### Underlayment & Protection
- [ ] **Starter shingle** - Most commonly missed item
- [ ] **Ice & water shield** - Required in cold climates per IRC R905.2.7.1
- [ ] **Synthetic underlayment** OR **Felt (15# or 30#)**

### Edge Protection
- [ ] **Drip edge - Eave** - Required per IRC R905.2.8.5
- [ ] **Drip edge - Rake** - Required per IRC R905.2.8.5

### Valleys & Flashing
- [ ] **Valley lining** - Metal or ice & water
- [ ] **Step flashing** - At wall intersections
- [ ] **Counter flashing** - At masonry

### Ventilation
- [ ] **Ridge vent** OR **Turbine vents** OR **Box vents**
- [ ] **Intake ventilation** - Soffit or edge vents

### Penetrations
- [ ] **Pipe jacks/boots** - All plumbing penetrations
- [ ] **Exhaust caps** - Kitchen, bath, dryer vents

### Caps & Finishing
- [ ] **Hip & ridge caps** - Cut shingles or manufactured

---

## ðŸŸ¡ CONDITIONAL ITEMS (Check If Applicable)

### Roof Characteristics
- [ ] **Steep slope charge** - If 7/12 pitch or greater
- [ ] **High roof charge** - If 2+ stories
- [ ] **Cut-up roof charge** - Complex roof with many facets

### Existing Conditions
- [ ] **Tear-off labor** - If not new construction
- [ ] **Haul-off/disposal** - Dumpster or dump fees
- [ ] **Additional layer removal** - If multiple existing layers

### Chimneys & Skylights
- [ ] **Chimney flashing** - Cricket if chimney > 30" wide
- [ ] **Chimney counter flashing**
- [ ] **Skylight flashing kit**
- [ ] **Skylight replacement** - If damaged

### Detach & Reset Items
- [ ] **Satellite dishes** - D&R
- [ ] **Roof-mounted antennas** - D&R
- [ ] **Solar panel D&R** - Requires licensed electrician
- [ ] **HVAC equipment D&R** - Roof-mounted units
- [ ] **Gutter D&R** - If necessary for roof work
